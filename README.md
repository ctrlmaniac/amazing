# Prima di cominciare

Che tu abbia forcato/scaricato questo repository o tu stia utilizzando docker, crea un file `.env` nella root del progetto o accanto al tuo file `docker-compose.yaml` con le seguenti variabili:

```
MYSQL_ROOT_PASSWORD=
MYSQL_DATABASE=
MYSQL_USER=
MYSQL_PASSWORD=

DATABASE_URL=
SECRET_KEY=
```
